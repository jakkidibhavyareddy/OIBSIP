# Exploratory Data Analysis (EDA) on Retail Sales Data

## Oasis Infobyte Data Analytics Internship - Task 1

### Project Overview
This project performs Exploratory Data Analysis (EDA) on the Superstore dataset using Python.

### Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

### Analysis Performed
- Data Inspection
- Missing Value Analysis
- Descriptive Statistics
- Monthly Sales Trend
- Top 10 Best-Selling Products
- Sales by Category
- Profit by Region
- Customer Segment Analysis
- Correlation Heatmap

### Business Insights
- Identified the highest-selling products.
- Compared sales across categories.
- Analyzed regional profit.
- Studied customer segments.
- Found relationships between Sales, Profit, Quantity, and Discount.
