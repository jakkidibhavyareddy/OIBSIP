# 🏠 House Price Prediction using Linear Regression

## 📌 Project Overview
This project predicts house prices using a Linear Regression Machine Learning model. The dataset contains information such as area, bedrooms, bathrooms, stories, and other house features.

## 🎯 Objective
To build a regression model that predicts house prices based on different house characteristics.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## 📊 Project Workflow
1. Data Loading
2. Data Exploration
3. Data Cleaning
4. Feature Encoding
5. Train-Test Split
6. Linear Regression Model
7. House Price Prediction
8. Model Evaluation

## 📈 Model Performance
- Mean Absolute Error (MAE): 1,029,305.71
- Root Mean Squared Error (RMSE): 1,413,204.62
- R² Score: 0.6049

## 💡 Key Insights
- House area has a strong impact on price.
- More bathrooms and stories generally increase house prices.
- Air conditioning and main road access positively affect the predicted price.
- The model explains approximately 60% of the variation in house prices.
